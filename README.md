# [Vue + Express Crash Course](https://github.com/thulin82/vue-express-crash-course)

Setup
---------------------
### Get project
```
$ git clone git@github.com:thulin82/vue-express-crash-course.git
```
### Get dependencies
```
$ npm install
```
### Run app
```
$ npm start
```
